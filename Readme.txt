Installation
------------

1. Unzip pk-flickr-recent-photos.zip
2. Copy the folder to /wp-content/plugins/ folder of Wordpress
3. Activate it (PK Flickr Recent Photos) in Plugins section of Wordpress admin page
4. Go to Presentation and then Widgets
5. Finally, add the widget to a sidebar



Settings
--------

- Title: any name you like to display as a widget title
- Username: is your flickr username
- Number of photos: a number of photos you want to display
- Width: the width of the display photos
- Height: the height of the display photos
